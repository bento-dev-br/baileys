<br/>

### 🏠 [Homepage](https://spider.aldebarandev.com)

<br/>

## Install on Linux/Termux

```sh
bash install.sh
```

## Usage in production

```sh
npm start
```

## Usage in development

```sh
npm run dev
```

## Run tests

```sh
npm test
```
